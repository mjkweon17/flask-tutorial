### 가상환경 생성 및 Flask 설치

- 아래 명령어로 가상 환경 폴더 만듬
    - python3 -m venv .venv
- 가상환경 활성화 명렁어
    - .venv\Scripts\activate.bat
- Flask 설치
    - pip install Flask
